# DataModel - DatabaseModeller for Laravel

Get's databases table information

### Requirements

- DataModel works with PHP 5.6 or above.
- DataModel only works with PostgreSQL and MySQL.

### Installation

```bash
$ composer require gguney/datamodel
```
